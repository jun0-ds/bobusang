Example rules:

Don't add features I didn't ask for. A bug fix is just a bug fix.
**Why:** Scope creep wastes time and introduces new bugs.
**How to apply:** Before making any change, check if it was requested.

Show me the code, not the explanation.
**Why:** I learn faster by reading than by listening.
**How to apply:** Default to code blocks. Add explanation only if asked.
